## Introduction

Ringo Pro is a application for music supervisors. It makes looking for tracks and searching for licenses easier.

## Usage

```zsh
git clone https://github.com/Ringo-Pro/Ringo.Pro.git
cd Ringo.Pro
npm install
npm start
```
